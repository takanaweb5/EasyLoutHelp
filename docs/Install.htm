インストール方法

(1)アドインに登録する方法

[ツール(T)] → [アドイン(I)] → [参照(B)] で表示されるフォルダに

以下の2ファイルを、コピーします

・EasyLout.xla

・EasyLout.chm

 [ツール(T)] → [アドイン(I)] → [参照(B)]で上記のEasyLout.xlaファイルを選択します

「かんたんレイアウト」ツールバーが表示されるので、インストール完了


(2)インストール時にセキュリティ関連のエラーが表示される時

セキュリティレベルが「高」に設定されているため、このままではインストール出来ません

※セキュリティレベルを変更する方法（セキュリティレベルの変更は各自の責任で行って下さい）

[ツール(T)] → [マクロ(M)] → [セキュリティ(S)]を選択します

セキュリティレベルを変更してください

アンインストール方法

(1)アドインを解除する方法

[ツール(T)] → [アドイン(I)]で「かんたんレイアウト」のチェックを外し、「OK」クリックします

以下の2ファイルを、アドイン用のフォルダより削除します

・EasyLout.xla

・EasyLout.chm

[ツール(T)] → [アドイン(I)]で「かんたんレイアウト」が存在していなければ完了、存在すればクリック

「ファイルが存在しない」というエラーが表示されてリストから削除されます

以上でアンインストール完了
