インストール方法

 

アドインに登録する方法

アドインに登録すれば使用出来るようになります。

[ファイル]タブ → [オプション] → [アドイン]

[設定(G)]をクリック

[参照(B)]をクリックして開いたアドイン用のフォルダに以下のファイルをコピー(ファイルをドラッグすればコピーできます)して選択します。

EasyLout.xlam

[有効なアドイン(A)]に「かんたんレイアウト」が出現しますので、チェックボックスにチェックされていることを確認して[ＯＫ]をクリックします。

[かんたんレイアウト]のタブが出現すればインストール完了です。

 

 

アンインストール方法

インストール時と同じ方法で以下の画面を起動します。

「かんたんレイアウト」のチェックを外し、以下のファイルをアドイン用のフォルダより削除します。

EasyLout.xlam

左側のリストから「かんたんレイアウト」の項目が表示されなくなればアンインストール完了です。
